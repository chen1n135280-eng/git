# Card Patterns

## Journal Entry

- Module 1: distinguish the accounting stage, account nature, or direction.
- Module 3: ask for account, measurement amount, and timing.
- Module 4: explain each line and each wrong choice.
- Module 5: show one complete entry, replace it with changed numbers, then ask for key accounts and amounts.
- Module 6: compare adjacent stages or similar entries.

## Concept

- Module 1: distinguish the concept from its nearest neighboring concept.
- Module 3: classify a concrete numerical business fact.
- Module 4: explain the classification boundary.
- Module 5: show one classification example, then replace it with a changed entity or instrument.
- Module 6: test scope or exclusion.

## Recognition Or Measurement

- Module 1: identify the first condition or measurement basis.
- Module 3: ask which conditions are met and which amount is used.
- Module 4: show the calculation and explain rejected bases.
- Module 5: change the decisive condition and numbers.
- Module 6: compare the rule with its exception.

## Presentation Or Comprehensive Treatment

- Module 1: identify the destination or whether later reclassification is allowed.
- Module 3: classify an item before showing the presentation result.
- Module 4: explain current and subsequent treatment.
- Module 5: replace the item and amount, then ask for destination and later treatment.
- Module 6: compare the closest presentation category.
