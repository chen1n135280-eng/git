# CPA Knowledge Extractor

Create the smallest unit that can be explained, reviewed, and cited independently.

## Workflow

1. Read only page-numbered textbook chunks.
2. Split content into these categories: concept, recognition condition, measurement rule, accounting treatment, journal entry, presentation, exception, or mistake.
3. Separate a general rule from each material exception.
4. Merge synonymous points only when their conditions and accounting consequences match.
5. Preserve the narrowest supporting page range for every point.
6. Add prerequisite titles only when understanding genuinely depends on them.
7. Omit unsupported conclusions. Never manufacture page numbers or standard requirements.
8. Return structured knowledge-card data for difficulty rating and explanation.
